# FrontEnd-Master Blog Project

A simple and clean static blog website built with HTML.

This project currently includes a BLACKPINK-themed home page and a personal About page to showcase content structure, page linking, and basic front-end layout practices.

## Project Overview

The goal of this project is to practice:
- Building multi-page static websites
- Structuring semantic HTML content
- Organizing media assets for blog posts
- Creating easy navigation between pages

## Features

- Multi-page setup (`index.html` and `about.html`)
- Blog-style article layout
- Image-based content sections
- Internal page navigation links
- Beginner-friendly, lightweight structure

## Tech Stack

- HTML5

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Shreyash2246/FrontEnd-Master.git
   ```
2. Open the project folder.
3. Launch `blog/index.html` in your browser.
